# 太阳系模拟 (Solar System Simulation)

一个使用 HTML, CSS 和原生 JavaScript 构建的视觉精美且可交互的太阳系模拟项目。

## 功能特点

- **真实轨道模拟**：行星根据真实世界的相对速度绕太阳公转。
- **视觉细节**：
  - 每个行星都有独特的颜色和渐变效果。
  - 土星拥有标志性的光环。
  - 发光的太阳效果。
  - 带有闪烁星星的深空背景。
- **地月系统**：地球拥有一个围绕其旋转的月球，同时月球轨道环相对于观察者保持稳定（不随地球自转而翻滚）。
- **小行星带**：在火星和木星之间有一个动态生成的小行星带，包含数百个散布的小行星。
- **交互元素**：
  - 悬停在行星上可查看详细信息。
  - **速度控制**：通过滑块调整模拟速度。
  - **标签切换**：显示或隐藏行星名称。
- **多语言支持**：完全支持 **英语** 和 **中文**。通过切换按钮即时切换语言。

## 使用技术

- **HTML5**：语义化结构。
- **CSS3**：
  - 使用关键帧（Keyframes）实现轨道和旋转动画。
  - 使用 CSS 变量方便主题定制。
  - 采用“零尺寸包装器（Zero-Size Wrapper）”模式的高级定位技术，确保轨道完美对齐。
- **JavaScript (ES6+)**：
  - 动态生成星星和小行星。
  - 管理语言状态和 UI 控制。
  - 处理交互事件。

## 如何运行

1. 克隆或下载此仓库。
2. 在任何现代浏览器（Chrome, Firefox, Safari, Edge）中打开 `index.html` 文件。
3. 享受模拟体验！

## 技术亮点

### 月球轨道修复
本项目中一个有趣的挑战是确保月球轨道的显示正确。由于地球绕太阳公转（并且自身也在自转），简单地将月球作为地球的子元素会导致其轨道随地球一起晃动或旋转。
我们的解决方案是：
1. 使用 **反向-反向旋转（Counter-Counter-Rotation）** 技术：月球轨道环应用一个反向旋转动画来抵消地球的公转旋转，使轨道环在屏幕坐标系中保持固定。
2. 将月球的公转分离到一个独立的 `moon-container` 中，该容器在固定的轨道环内独立旋转。

### 小行星带
小行星带是通过 JavaScript 动态生成的，以创建具有不同大小、透明度和位置的自然散布外观，而不是使用静态图片。
